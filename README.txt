README

In order to run the program follow these instructions:

1. Copy the following files to home directory of current user in storm.cise.ufl.edu:
	- report.txt
	- start.jar

2. Execute: java -jar start.jar

Step 2 will run the application and produce console output and log files in the user's home directory, with names: ID1 through ID5. At this point the script to cat all the logs can be executed
