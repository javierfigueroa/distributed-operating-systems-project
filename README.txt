README

In order to run the program follow these instructions:

1. Copy all files to a desired location:
	- project1.tar
	- report.txt
	- start.jar

2. Execute: java -jar start.jar

Step 2 will run the application and produce console output and log files in the user's home folder, with names: ID1 through 5. At this point the script to cat all the logs can be executed
